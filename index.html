PoshProgressBar

This is a project intended to make it easy to add beautiful GUI based Progressbars to any PowerShell script or project

Installation Methods

  1. Download and install the module
  2. Install from the PowerShell Gallery with the following code:

Install-Module -Name PoshProgressBar

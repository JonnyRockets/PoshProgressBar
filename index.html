PoshProgressBar
A PowerShell ProgressBar in XAML using Material Design in XAML Toolkit from ButchersBoy

Read the four part blog series
Installation Methods
  • Download from Github
  • PowerShell Gallery
Usage


Step 1: Create your progress bar
                    $ProgressBar = New-ProgressBar
Step 2: Write your progress
                    Write-ProgressBar -ProgressBar $Progressbar `
                        -Activity "Prettyin up Powershell" `
                        -PercentComplete 50 `
                        -Status "Prettyin" `
                        -SecondsRemaining 100 `
                        -CurrentOperation "Progressifyin"
Step 3: Gracefully close out that beautiful progressbar
                    $ProgressBar = New-ProgressBar
